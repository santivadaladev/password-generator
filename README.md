Generatore di Password Sicure
Descrizione

Questo progetto è un generatore di password sicure creato con HTML, CSS e JavaScript.
Permette di generare password casuali in base a lunghezza e tipologia di caratteri scelti: maiuscole, numeri e simboli.

È un progetto veloce e semplice, perfetto per chi vuole vedere subito risultati e imparare a lavorare con DOM, eventi e manipolazione dei dati in JavaScript.

Funzionalità

Selezione della lunghezza della password (da 4 a 64 caratteri)

Opzioni per includere:

Maiuscole
Numeri
Simboli
Generazione password casuale con un click
Copia negli appunti con un click
UI semplice e responsiva
